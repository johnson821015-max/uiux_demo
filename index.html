什麼是 Design Token？
Design Token 是設計系統的最小單位——顏色、字級、間距等，全部集中管理，改一處全站生效。
如何用這個模板改風格？
只要修改 :root 內的 CSS 變數，所有元件會自動套用新風格。
可以用在 App 上嗎？
可以，下方的手機框架區展示了 App 常用元件，AI 可以把這些 tokens 轉換成 Flutter / React Native 的主題設定。
